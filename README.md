# Water-Reminder

###  饮水提醒，一个贴心的应用程序，可以照顾你的健康。如果你太忙而忘记了要按时喝足够的水，别担心，这里有“饮水提醒”来帮你解决这个问题。饮水提醒是一个应用程序，其主要功能是帮助我们保持水分。用户只需要选择性别并输入体重数字，它就会帮助你计算出每天应该喝多少水。您还可以跟踪饮水的历史，实现您的日常目标，


# 工具 
- Android Studio

# Functionalities
- 易于使用，美观的界面。
- 根据性别，体重会告诉你一天应该喝多少水。
- 近20种不同饮品的选项。
- 可以选择每次的水量。
- 智能提醒:在你的睡觉时间不会发出饮水提醒。
